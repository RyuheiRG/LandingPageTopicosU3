frame-data-bayes

Desarrolle Frame-Data-Bayes, el cual es un motor de análisis probabilístico y clasificación estadística desarrollado bajo estándares de Clean Code y mentalidad Zero Trust. El sistema trata cada dataset como un entorno hostil: valida superficies de entrada, detecta tipos de datos de forma determinística y aplica inferencia bayesiana para predecir eventos anómalos.
