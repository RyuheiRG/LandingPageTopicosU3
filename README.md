# Landing Page

Landing Page desarrollada como parte de un proyecto escolar, cuyo objetivo fue personalizar una plantilla base con información personal, proyectos académicos, tecnologías dominadas y eventos relevantes, aplicando buenas prácticas de estructura HTML, diseño responsivo y organización de recursos.

## 📖 Descripción

El proyecto consiste en una página web estática tipo **Landing Page** enfocada en presentar un perfil profesional de desarrollador.

La página fue adaptada con información personalizada, incluyendo:

- Presentación personal.
- Tecnologías y lenguajes de programación.
- Proyectos destacados.
- Participación en eventos académicos.
- Certificaciones y cursos.

Además, se implementó un diseño responsivo para una correcta visualización en equipos de escritorio, tabletas y dispositivos móviles.

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design
